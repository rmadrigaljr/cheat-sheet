Selector Name Description

Specificity

Priority Name Description
1st id attribute This is the most specific way to target an element.
2nd class attribute Elements can have many classes, therefor are not as specific as the id attribute.
3rd h1, h2, p, etc... Since elements can be used multiple times on a web page, they are the less specific way to target an element.
